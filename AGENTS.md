## Project instructions

- Always use `next-devtools` MCP for Next.js diagnostics and runtime checks.
- Use `tailwindcss` MCP when generating or validating utility class combinations.
- Use `shadcn` MCP to reference component patterns before creating custom UI.
- Use `bun` MCP for Bun runtime, package, and script-related tasks.
- Use `eslint` MCP to inspect lint issues and enforce formatting via ESLint rules.
- Use `lucide` MCP to search and pick icons from the Lucide set.
